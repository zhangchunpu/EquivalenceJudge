
# ルール
1. 変数名について
 - 変換ならA_to_B
 - n個の変数を扱う場合、nを変数名の最後につける。
 - 変数の型を変数名に含める。nの前。明示的にわかるものは書かない。
 - 必要最低限の情報を入れる。
    - theは不要。
    - equation -> eq
    - variable -> var
    - function -> func
    - list は list.
